# Personal Portfolio Website

## Overview

This is my personal portfolio website developed using HTML and CSS. The website showcases my profile, qualifications, skills, achievements, certificates, resume, and academic projects.

The portfolio is designed with a clean and modern user interface and is fully responsive for different screen sizes.

---

## Features

* Responsive Design
* Modern Navigation Bar
* Home Page with Introduction
* About Page
* Resume Page
* Projects Page
* Email, Phone, and LinkedIn Links
* Downloadable Resume
* Project Cards with Images
* Direct Links to Project Demonstrations

---
